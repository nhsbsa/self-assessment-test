Content Designer Skill Assessment

This tool was built referencing the Government Digital and Data Profession Capability Framework.

It assesses skill level across seven core areas, using multiple choice questions

Content Designers can use this tool to:

  • assess their current skill levels in preparation for performance and development conversations
  • learn about the typical responsibilities and skills of their colleagues

Before you start, we recommend you:

  • set aside five to eight minutes to complete this test in one seating
  • be as objective as you can when answering the questions
  • remember that this is to aid your self evaluation and professional development, it is not a tool to request a change to contract or working conditions
